# English IPA Learning for Chinese Learners / 面向中文学习者的英语音标学习网页

这是一个正在尝试制作中的英语音标学习网页。

我本人并不懂英语，也不懂编程，只是想试着做一个网站，或者以后做成一个 app，用来帮助自己学习英语音标。如果它也能帮助到其他有需要的人，那就更好了。

目前这个项目还很不完善，里面的音标、单词、释义、发音、交互设计都可能有错误或不准确的地方。

如果你懂英语、英语发音、音标教学，或者懂前端编程，欢迎帮忙完善：

- 修正音标和单词 IPA
- 补充更准确的中文释义
- 增加标准真人发音音频
- 改进适合中文学习者的发音提示
- 优化网页交互和界面
- 帮助整理词库、音频和练习题

谢谢每一个愿意帮忙的人。

## Current Features / 当前功能

- 英语音标卡片
- 代表词、完整 IPA、中文释义
- 单词点击发音
- 音标点击发音
- 易混淆音标学习
- 听辨练习
- 试卷练习
- 本地音频文件夹结构

## Usage / 使用方式

直接打开 `index.html` 即可使用。

本地音频可以放在 `audio` 文件夹下，具体命名参考 `audio/audio-manifest.csv`。
